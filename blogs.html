Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are multiple ways to select elements from the DOM. Such as - getElementById, getElementsByClassName, getElementsByTagName, getElementsByName, querySelector, querySelectorAll etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHtml: Returns or sets the HTML content of an element, including nested HTML tags. innerText: Returns or sets the visible text of an element. textContent: Returns or sets all text inside an element, including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation in JavaScript is a method where a single event listener is assigned to a parent element rather than adding multiple listeners to each child element. This approach covers event bubbling, allowing events triggered on child elements to propagate up the DOM hierarchy to their ancestors.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event that occurs on a child element first runs its associated event handlers. Then, it moves upward through its parent elements, eventually reaching the root.

Question-5: How do you create, add, and remove elements using JavaScript?

To create new element we use like document.createElement('div'). To add elements to the DOM, we use like appendChild('p'). To remove elements from the DOM we simply use remove().
